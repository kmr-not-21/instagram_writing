# 通常投稿 執筆ルール

インスタグラムアカウント：https://www.instagram.com/hana.moja_/
店のコンセプト：「花とグリーン、どちらも主役級。特別なかけあわせ。」

---

## 1. 全体構成（3〜4段落 + フッター + ハッシュタグ）

投稿本文は以下の順序で構成する。

### 第1段落：キャッチコピー（1行）

- **体言止め**で、印象的かつ詩的な一文にする。
- 句点「。」で終える。
- テーマの花・グリーンの個性と、投稿全体のトーンを凝縮する。
- 例：「手のひらサイズのちょこんと早春。球根と苔の芽吹きアレンジメント。」

### 第2段落：主役の花・グリーンの魅力

- トップ写真に写っている主役の花やグリーンを具体的に紹介する。
- **品種名**を正式に記載する（例: スイートピー、チューリップ"ホワイトフラッグ"）。
- **色・質感・形・ライン**など、五感に訴える描写を入れる。
- 花言葉やその花の特性など、知識的な要素を入れてよい。
- 1枚目の写真の内容と紐づけて描写する（「写真1枚目の…」「1枚目の花束には…」）。

### 第3段落：かけあわせの魅力 / テーマの深掘り

- 複数の花とグリーンの**組み合わせ（かけあわせ）**の妙を伝える。
- コンセプトである「花とグリーン、どちらも主役級。特別なかけあわせ。」を体現するパート。
- 以下のいずれかの切り口で書く：
  - 組み合わせによって生まれる**コントラスト・調和・奥行き**
  - 技術的なこだわりや制作意図
  - テーマに関する深い知識やストーリー
  - 花の品種・咲き方のバリエーション紹介（知識共有型）
- テーマによっては、第2段落でアレンジメント全体の描写をまとめ、第3段落で花の知識共有（品種の多彩さ、開花による変化など）に充てる構成も有効。
- **段落間のつながり**を意識する。視点を自然にスライドさせる（例：品種間の多様さ → 一輪の中での開花変化）。
- 🌱 のような**絵文字は段落末尾に1つだけ**使ってよい。

### 第4段落：利用シーン提案 + 来店誘導

- 具体的な利用シーン（ギフト、イベント、ご自宅用など）を提案する。
- **季節行事**（成人の日、卒業・入学、母の日など）があれば自然に関連づける。
- 「ぜひMojaで〜」「ぜひ実物で〜」のように来店を促す。
- **`(*^^*)`** で文末を締める（毎回必須）。

### フッター（固定テンプレート）

```

『花とグリーン、どちらも主役級。特別なかけあわせ。』
花屋Moja

所在地：神奈川県藤沢市下土棚260-9
営業時間：9:00～18:00(水曜のみ13:00まで・日曜定休)

※不定期のお休みや時間変更については、カレンダーの投稿をご確認くださいませ。

---------
```

### ハッシュタグ

- フッターの後に1行で記載する。
- **5個**とする。
- 必ず含めるもの：`#藤沢花屋` `#湘南花屋`
- テーマに応じて追加：花の名前、シーン（`#卒業祝い`）、特徴（`#花のある暮らし` `#フラワーギフト`）。

## 2. 文体・トーンのルール

- **です・ます調**を基本とする。
- **体言止め**をキャッチコピーだけでなく、段落内でも効果的に使い、テンポを出す。
- **比喩・感覚的表現**を積極的に使う（「まるでドレスのような華やかさ」「ふわっと空間を作る」など）。
- **五感に訴える描写**を重視する：色あい、質感（ふわふわ、しっとり、瑞々しい）、香り、形・ラインなど。
- 堅すぎず、柔らかく知的な語り口。**店主が語りかけるような親しみやすさ**を保つ。
- 過度な装飾語や感嘆を避け、**品のある抑制の効いたトーン**にする。
- 絵文字は🌱など自然系のものを**1投稿に1〜2個**に抑える。

---

## 3. コンテンツの注意事項

- **過去に使った文章をそのまま引用・再利用することは禁止。**文体や構成のみを参考にする。
- 写真に含める要素として、トップ写真において特徴的な要素を「植物の質感」「色あい」「形やライン」「季節性(現在日時を参照)」「組み合わせ」の中からピックアップする。
- **季節性**は現在日時を参照し、旬の花・季節行事と自然に結びつける。
- 品種名が不明な場合はプレースホルダー `{{品種名}}` を置き、確認を求める。
- **投稿文面のファイル形式は `.txt`（プレーンテキスト）とする。** `.md`（マークダウン）では `(*^^*)` などの記号がマークダウン記法として解釈され、意図しない表示になるため。
- **写真の構成を正確に把握すること。** 器が複数ある場合はそれぞれの中身を個別に確認し、花材の組み合わせを思い込みで決めつけない。
- **テーマのキーワードや表現は事前に検討すること。** 新しい概念や造語を使う場合は、一般的な理解のされ方・誤解の可能性・代替表現を検討し、ユーザーに確認を取ってから執筆に入る。
