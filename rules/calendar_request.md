# カレンダー投稿リクエストテンプレート

以下のルールに従って、インスタグラムに投稿する **カレンダー（営業日案内）投稿** の文面を **2パターン** 提案してください。

## 参照ルール

- 執筆ルール：`rules/calendar_writing_rules.md`
- 投稿文面の出力先：`.txt`（プレーンテキスト）形式で作成すること

## 投稿情報（毎回記入）

- **対象月**：{{対象月を記入（例：2026年3月）}}
- **不定期休業日**：{{不定期の休業日を記入（例：3月5日(水)・6日(木)、3月20日(木)）}}
- **今月の入荷・見どころ**：{{その月に店頭に並ぶ旬の花やグリーン、作品の情報を記入（例：ミモザ、ラナンキュラス、ドライフラワーリースなど）}}
- **季節行事・イベント**：{{任意。ホワイトデー、卒業シーズンなど、関連する行事があれば記入}}
- **写真の説明**：{{カレンダー画像の他に添付する写真があれば記入}}
