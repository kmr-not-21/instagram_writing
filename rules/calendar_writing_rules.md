# カレンダー投稿 執筆ルール

インスタグラムアカウント：https://www.instagram.com/hana.moja_/
店のコンセプト：「花とグリーン、どちらも主役級。特別なかけあわせ。」

---

## 1. 全体構成（3〜4段落 + フッター + ハッシュタグ）

カレンダー投稿は営業日案内を兼ねた月次投稿。花の魅力を伝えつつ、実用的な情報を自然に織り込む。

### 第1段落：季節の花の紹介（軽め）

- その月に入荷している旬の花・グリーンを**2〜3種類**さらりと紹介する。
- 季節感を伝える描写を入れるが、通常投稿ほど深掘りしない。
- 「〜が楽しくなってくる時期です」のような柔らかい語り口。

### 第2段落：営業日・休業日のお知らせ

- **不定期休業日**を具体的な日付と曜日で明記する。
- 簡潔に、丁寧な言い回しで（「〜となりますので、ご来店の際はご確認くださいませ」）。

### 第3段落：今月の見どころ + 来店誘導

- 今月の店頭に並ぶ商品や作品の見どころを軽く紹介する。
- **ギフト需要**や**ご自宅用**の提案を自然に盛り込む。
- 🌱 のような絵文字を**1つ**使ってよい。
- **`(*^^*)`** で文末を締める（毎回必須）。

### フッター（固定テンプレート）

通常投稿と同じフッターを使用する。

```
---------

『花とグリーン、どちらも主役級。特別なかけあわせ。』
花屋Moja

所在地：神奈川県藤沢市下土棚260-9
営業時間：9:00～18:00(水曜のみ13:00まで・日曜定休)

※不定期のお休みや時間変更については、カレンダーの投稿をご確認くださいませ。

---------
```

### ハッシュタグ

- **5個**とする。
- 必ず含めるもの：`#藤沢花屋` `#湘南花屋`
- 季節やその月の特徴に合わせて選ぶ。

---

## 2. 文体・トーンのルール

- 通常投稿と同じ **です・ます調** を基本とする。
- 通常投稿に比べて**軽やかで簡潔**なトーン。詩的な深掘りは控えめに。
- 体言止めはキャッチコピー的な要素には使ってよいが、全体的には情報伝達を優先する。
- 親しみやすく、月の始まりのワクワク感を演出する。

---

## 3. コンテンツの注意事項

- **過去に使った文章のそのまま引用・再利用は禁止。**
- 営業日情報は正確に記載すること（必ず確認を求める）。
- 季節性は現在日時を参照し、旬の花や行事と自然に結びつける。
- **投稿文面のファイル形式は `.txt`（プレーンテキスト）とする。** `.md`（マークダウン）では `(*^^*)` などの記号がマークダウン記法として解釈され、意図しない表示になるため。
- **写真がある場合は構成を正確に把握すること。** 思い込みで内容を決めつけず、不明点は確認する。
- **テーマのキーワードや表現は事前に検討すること。** 新しい概念や造語を使う場合は、一般的な理解のされ方・誤解の可能性・代替表現を検討し、ユーザーに確認を取ってから執筆に入る。
