# 通常投稿リクエストテンプレート

以下のルールに従って、インスタグラムに投稿する花屋の告知文面を **2パターン** 提案してください。

## 参照ルール

- 執筆ルール：`rules/writing_rules.md`
- 投稿文面：`draft/{{投稿日時}}/content.txt`

## 投稿情報（毎回記入）

- **投稿のテーマ**：{{テーマを記入（例：春のチューリップ、母の日ギフトなど）}}
- **写真の説明**：
  - トップ（1枚目）：{{トップ写真の内容を記入（例：添付一枚目のピンクのチューリップの花束）}}
  - その他：{{2枚目以降の写真の内容を記入（例：アレンジメント、店内の様子など）}}
- **主な花材・グリーン名**：{{花材名を記入（例：チューリップ"ホワイトフラッグ"、ユーカリなど）。不明な場合は「写真から判断してください」}}
- **アピールしたいポイント**：{{任意。特にこだわりや伝えたい要素があれば記入}}
