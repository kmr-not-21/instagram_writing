# 通常投稿リクエストテンプレート

あなたは、花業界のマーケティングに精通した熟練のWebライターです。
以下のルールに従って、インスタグラムに投稿する花屋の告知文面を **2パターン** 提案してください。
いずれかのステップで私の説明や意見が必要な場合は、[Question]タグを付けて質問ドキュメントを下部に追加し、推奨案を添えて、私が回答を記入するための空の[Answer]タグを作成して人間にボールを回してください。重要な決定を独自に行わないでください。

## 参照ルール

- 執筆ルール：`rules/writing_rules.md`
- 投稿文面案：`draft/{{投稿日}}/content.txt`

## 作業の進め方

文面を書き始める前に、以下のステップを踏むこと。

### ステップ0：過去投稿の確認
- `posted/normal_post/` にある過去の確定投稿を読み、文体・構成・トーンを把握してから執筆に入る。

### ステップ1：写真の構成を正確に把握する
- トップ写真に写っている要素（花材、器、配置、数量）を正確に読み取り、[Question]で確認する。
- 器が複数ある場合、それぞれに何が入っているかを個別に把握する。
- 思い込みで花材の組み合わせを決めつけない。

### ステップ2：テーマの表現分析・検討
- いきなり文面を書き出さず、テーマを魅力的に伝えるための**表現やキーワードを分析・検討する**ステップを挟む。
- 以下の観点で検討し、request.mdに[Question]として提示する：
  - テーマを端的に伝えるキーワード候補（例：「二輪挿し」「ペアリング」「2本生け」など）
  - そのキーワードの適切さ（一般的な理解、誤解の可能性、業界での使われ方）
  - キャッチコピーの方向性（詩的・提案型・How to型など）
  - テーマの魅力を伝える切り口（コツ提案、ライフスタイル提案、知識共有など）
  - プロとしての視点、工夫(花選びや、生け方、組み合わせ、色合わせ、質感、ラインなど)

### ステップ3：文面の執筆
- ステップ1・2の確認が取れてから、`rules/writing_rules.md` に従って文面を執筆する。

## ファイル運用ルール

- `draft/{{投稿日}}/content.txt` には検討過程（全パターン）をそのまま残す。クリーンアップしない。
- 確定版は `posted/normal_post/{{投稿日}}.txt` に生成する。
- **日付のフォーマットはハイフンつなぎ `YYYY-MM-DD` で統一する。**（例：`2026-03-07.txt`）

### [Question 2] 「結婚式の前撮り用途」への言及

トップのクラッチブーケは結婚式の前撮り用とのことですが、本文でこの文脈を打ち出しますか？

▼選択肢：
- **A. 前撮り用途を前面に出す** — 「人生の節目を彩るブーケ」「ハレの日に寄り添う」など、結婚式・前撮りシーンを軸にした切り口。第4段落の利用シーン提案も「前撮り・結婚式・記念日」方向へ。
- **B. 前撮り用途は伏せ、初夏×白×新緑の色合わせとして語る** — 用途を限定せず、初夏のシャクヤクの魅力やかけあわせの妙を主軸に。第4段落は「初夏のギフト・自分用・お祝い事全般」へ。

**推奨案:** **C（中間）**。前撮りという華やかなシーンは差別化要素になりますが、Mojaの読者層は日常使い・ギフト需要も多いため、用途を狭めすぎないバランスがよいと判断しました。第3段落までは「初夏×白×新緑」の色合わせを主軸に、第4段落の利用シーン提案で前撮り・記念日を自然に滑り込ませる構成です。

### [Question 5] テーマの切り口・キャッチコピー方向性

テーマ「初夏、シャクヤク」「新緑とのかけあわせ」を魅力的に伝える切り口を検討しました。

▼切り口の候補：
- **(a) 色合わせ提案型** — 「白×新緑」の凛とした初夏らしい色合わせの妙を伝える（コンセプト「かけあわせ」直結）
- **(b) 知識共有型** — シャクヤクの品種・色・咲き進みの変化（つぼみ→大輪）を語る
- **(c) ライフスタイル提案型** — 前撮りや記念日の特別な一束として提案する

**推奨案:** **(a)を主軸＋(b)を第3段落で補完**。第2段落でトップ写真の白シャクヤク×新緑の凛とした表情を描写し、第3段落で「シャクヤクは白だけじゃない、つぼみから大輪まで一輪の中で表情が変わる」という知識を写真2の素材を使って広げる構成。

▼キャッチコピー方向性（5〜6案を比較検討）：

1. 「初夏の白に、新緑を抱いて。シャクヤクのクラッチブーケ。」（詩的・体言止め純度高）
2. 「白いつぼみと、芽吹きの緑。初夏のシャクヤク。」（対比型、二要素並列）
3. 「凛と咲く白、瑞々しく寄り添う緑。シャクヤクと新緑の束。」（対比型、動詞を変えた構成）
4. 「咲きはじめの白シャクヤク。新緑とかさねる、初夏のしらべ。」（提案型寄り）
5. 「白の余白に、新緑を。初夏のクラッチブーケ。」（短く詩的）
6. 「ふくらむ白、ひろがる緑。初夏のシャクヤク。」（対比型、動詞対比）

**推奨案:** **3番（凛と咲く白、瑞々しく寄り添う緑。シャクヤクと新緑の束。）** または **6番（ふくらむ白、ひろがる緑。初夏のシャクヤク。）**。
- 3番：対比軸（白／緑）に異なる動詞（咲く／寄り添う）を当てたルール準拠型。少し長め。
- 6番：対比軸に短い動詞対比（ふくらむ／ひろがる）。シャクヤクのつぼみがふくらむ様と新緑のひろがる様が呼応し、写真とも噛み合う。短くリズミカル。

2パターン提案では、書き出しのアプローチを変えるため**3と6の方向性で1案ずつ**作ろうと考えています（片方は花材から、片方は季節感から入る）。

[Answer 5]
推奨の通り。
